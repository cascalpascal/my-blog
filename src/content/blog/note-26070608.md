## 네트워크 전송 방식

### 패킷 교환방식

- 패킷 전송

- 데이터를 패킷 단위로 나누어 독립적으로 전송

- Layer 3(네트워크)에서 패킷을 처리하고 라우팅

- 각 패킷이 서로 다른 경로를 통해 전송될 수 있음

#### 데이터그램

- 비연결형 교환 방식

- 독립적인 패킷을 헤더와 함께 전송

- 순서 무관

#### 가상 회선

- 연결형 교환 방식

- 논리적인 회선을 설정하고 패킷을 전송

- 미리 연결

#### 비연결형 vs 연결형

- 연결 설정 불필요 vs 필요

- 패킷 독립적 vs 종속적

- 경로 패킷마다 다름 vs 고정 경로

- 순서 보장 안됨 vs 보장됨

- **IP**(Layer 3), **UDP**(Layer 4) vs **TCP**(Layer 4), **ATM**(Layer 2~3)


### 서킷 교환방식

- 연속적인 비트 스트림 전송

- 통신 전에 송수신자 간의 전용 회선 설정

- Layer 1~2에서 회선 설정

- 한 번 회선이 설정되면 통신이 끝날 때까지 해당 회선 독점 사용

- 대역폭 보장, 지연 예측 가능
