## 메서드 오버로딩
**오버로딩**: 같은 이름의 메서드를 매개변수나 타입, 또는 개수를 다르게 해 여럿 정의하는 것

```java
Static int num(int i) {
return i * 2;
}

Static int num(String s) {
return Integer.valueOf(s) * 2;
}
```
- 메서드 이름이 같아도 매개변수의 타입이나 개수가 다르면 서로 다른 메서드 취급
- Java가 전달하는 인자 타입에 따라 어떤 메서드를 호출할지 자동으로 결정함
- <code>num(10)</code> &rarr; <code>num(int i)</code> 호출
- <code>num("10")</code> &rarr; <code>num(String s)</code> 호출
