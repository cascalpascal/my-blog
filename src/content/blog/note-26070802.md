## 재귀 호출
**Recursion**: 메서드가 자기 자신을 다시 호출하는 것

```java
static int fn(int n) {
    if (n <= 1) return 1;
    return n * fn(n - 1);
}
```
- <code>if (n <= 1)</code>: 재귀를 멈추는 지점

- <code>return n * fn(n - 1)</code>: 자기 자신을 호출

- 기저 조건이 없는 경우 메서드가 끝없이 자기 자신을 호출해 Stack Overflow가 발생함

#### 펼쳐 내려가기
- <code>fn(3)</code> 호출
- <code>fn(3) &rarr; 3 * fn(2)</code>
- <code>fn(2) &rarr; 2 * fn(1)</code>
- <code>fn(1) &rarr; 1</code>

#### 거꾸로 계산하기
- <code>fn(1) = 1</code>
- <code>fn(2) = 2 * 1 = 2</code>
- <code>fn(3) = 3 * 2 = 6</code>

### 자기 자신을 두 번 호출하는 재귀

```java
static int fn(int n) {
    if (n <= 1) return n;
    return fn(n - 1) + fn(n - 2);
}
```

#### 펼쳐 내려가기
- <code>fn(4)</code> 호출
- <code>fn(4) = fn(3) + fn(2)</code>
- <code>fn(3) = fn(2) + fn(1)</code>
- <code>fn(2) = fn(1) + fn(0)</code>
- <code>fn(1) = 1</code>
- <code>fn(0) = 0</code>

#### 거꾸로 계산하기
- <code>fn(0) = 0</code>
- <code>fn(1) = 1</code>
- <code>fn(2) = 1 + 0 = 1</code>
- <code>fn(3) = 1 + 1 = 2</code>
- <code>fn(4) = 2 + 1 = 3</code>

### 구간을 나눠 호출하는 재귀

```java
static int divide(int[] a, int start, int end) {
    if (start >= end) return 0;
    int mid = (start + end) / 2;
    int left = divide(a, start, mid);
    int right = divide(a, mid + 1, end);
    return a[mid] + Math.max(left, right);
}
```
