# 정보처리기사 실기 

## 2026년 1회 기출 풀이

1. 다음은 C 언어에 대한 문제이다. 아래 코드를 확인하여 알맞는 출력값을 작성하시오.

~~~C
#include <stdio.h>
 
double arr1(int p[], int len) {
    double av = 0;
    int i;
    for (i = 0; i < len; i++) {
        av += (double) p[i];
    }
    return av / len;
}
 
double arr2(int * p, int len) {
    double av = 0;
    int i;
    for (i = 0; i < len; i++) {
        av += (double)( * (p + i));
    }
    return av / len;
}
 
int main() {
    int arr[10] = {
        80,
        20,
        50,
        55,
        45,
        95,
        55,
        10,
        40,
        80
    };
    int len = 10;
 
    printf("%.2f", arr1(arr, len) + arr2(arr, len));
 
    return 0;
}
~~~

**<code>main</code>**  
- 크기 10의 정수 배열 <code>arr</code> 초기화하고 <code>len</code>에 10 대입
- <code>arr1(arr, 10) + arr2(arr, 10)</code>의 값을 **소수점 둘째 자리**까지 출력  

<br>

**<code>arr1(arr, 10)</code>**  
-  <code>double arr1(int p[], int len)</code>: **C언어에서 배열 매개변수는 포인터**
- <code>p</code> &rarr; <code>arr[0]</code>의 주소, <code>len</code> &rarr; 10, <code>av</code> &rarr; 0.0
- <code>for</code> 반복문 실행  

| 회차 | <code>i</code> | <code>p[i]</code> | <code>av</code> |
|:---:|:---:|:---:|:---:|
| 1 | 0 | 80 | 80.0 |
| 2 | 1 | 20 | 100.0 |
| 3 | 2 | 50 | 150.0 |
| 4 | 3 | 55 | 205.0 |
| 5 | 4 | 45 | 250.0 |
| 6 | 5 | 95 | 345.0 |
| 7 | 6 | 55 | 400.0 |
| 8 | 7 | 10 | 410.0 |
| 9 | 8 | 40 | 450.0 |
| 10 | 9 | 80 | **530. 0**|

- <code>return av / len</code>: 530.0 / 10 = 53.0 반환

<br>

**<code>arr2(arr, 10)</code>**
- <code>double arr2(int *p, int len)</code>: <code>arr1</code>의 <code>int p[]</code>와 동일하게 포인터 <code>p</code> 선언
- <code>p</code> &rarr; <code>arr[0]</code>의 주소, <code>len</code> &rarr; 10, <code>av</code> &rarr; 0.0
- <code>for</code> 반복문의 <code>av</code>의 <code>*(p + i)</code>는 **C언어 포인터 산술**에 따라 <code>p[i]</code>와 같음
- 따라서 반환값도 <code>arr1</code>과 동일한 53.0 반환

**printf**
- <code>arr1(arr, len) + arr2(arr, len)</code>의 결과값 106.0을 <code>"%.2f%</code>에 의해서 소수점 아래 둘째자리까지 출력해서 **106.00**

6. HDLC는 비트 중심의 데이터 링크 제어 프로토콜로, 프레임 단위로 데이터를 전송하며 흐름 제어 및 오류 복구 기능을 제공한다. 다음 설명을 읽고 알맞은 용어를 쓰시오.

    HDLC 구성 요소  
    1. HDLC 프레임의 구성 단위로, 실제 사용자 데이터를 전송하는 프레임
    2. 데이터 링크의 흐름을 관리하고 오류 제어 및 통신 상태를 감시하는 프레임
    3. 순서 번호 없이 링크 설정, 해제, 모드 설정 등 제어 기능을 수행하는 프레임
    4. 두 국(Station)이 동등한 위치에서 서로 명령과 응답을 주고받는 모드
    5. 종국(Secondary)이 주국(Primary)의 허가 없이도 자발적으로 응답을 전송할 수 있는 모드
