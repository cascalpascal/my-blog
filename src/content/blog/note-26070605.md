## 비기능 요구사항 6가지 유형


| 유형 | 의미 |
| --- | --- |
| 품질(Quality) | 품질 속성(신뢰성·가용성·유지보수성) |
| 운영(Operational) | 어떤 환경(OS·플랫폼·언어 등)에서 동작 |
| 자원(Resource) | 하드웨어 자원(메모리·디스크) |
| 성능(Performance) | 처리 속도·응답성·동시 사용자 수 |
| 보안(Security) | 인증·인가·기밀성 |
| 문화/정책(Cultural/Political) | 사회문화적 제약(법규·관습·정책) |
