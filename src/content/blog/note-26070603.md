### C 언어 포인터 산술 
| 포인터 | 의미 | 배열 |
| :--- | :--- | :--- |
|  <code>*(p + 0)</code> | 배열의 인덱스 0 위치의 요소 | <code>p[0]</code> |
|  <code>*(p + 1)</code> | 배열의 인덱스 1 위치의 요소 | <code>p[1]</code> |
|  <code>*(p + i)</code> | 배열의 인덱스 i 위치의 요소 | <code>p[i]</code> |
| <code>*p + 1</code> | <code>p</code>가 가리키는 값에 1를 더한 값 |  |

### C 언어 구조체 포인터 산술

```c
struct Student {
    int num;
    char name[20];
};

struct Student s[] = {{1, "Blair"}, {2, "Jenny"}, {3, "Serena"}};
struct Student *p = s;
```
- <code>p</code>는 <code>s[0]</code>의 주소를 가리키는 포인터
- <code>p&rarr;num</code>: <code>s[0].num</code>과 같으며 결과는 1
- <code>(p + 1)&rarr;num</code>: <code>s[1].num</code>과 같으며 결과는 2
- <code>(p + 2)&rarr;name</code>: <code>s[2].name</code>과 같으며 결과는 "Serena"

```c
struct University {
    int i;
    const char *g;
};

struct University uni[] = {{1, "Harvard"}, {2, "Yale"}, {3, "NYU"}};
struct University *p = &uni[1];

printf("%s", p->g + (p->i - 1));
```
- <code>const char *g</code>: 내용 변경이 불가능한 문자열을 가리키는 포인터
- <code>p&rarr;i</code>가 2 이므로 <code>p&rarr;i-1</code> 값은 1
- <code>p&rarr;g</code>: 문자열 <code>"Yale"</code>의 시작 주소에서 1칸 이동한 <code>'a'</code>의 주소를 가리킴
- <code>"%s"</code>로 "ale" 출력

### 포인터 증감 연산자

```c
int a[] = {1, 2, 3, 4};
int *p = a;

printf("%d", *p);   // 1
p++;
printf("%d", *p);   // 2

int x = *p++;       // x = p가 가리키는 값 = 2, p는 3 주소로 이동
int y = (*p)++;     // y = p가 가리키는 값을 1 증가시킴 = 4
```
