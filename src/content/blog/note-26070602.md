### C 언어 ASCII 코드
컴퓨터가 문자를 직접 저장할 수 없기에, 각 문자에 숫자를 대응시킨 표준 코드를 **ASCII**라 한다.

| 문자 | ASCII 값 | 설명 |
| --- | --- | --- |
| <code>'\0'</code> | 0 | 널 문자 (문자열 끝 표시) |
| <code>'A'</code> | 65 | 대문자 시작 |
| <code>'Z'</code> | 90 | 대문자 끝 |
| <code>'a'</code> | 97 | 소문자 시작 |
| <code>'z'</code> | 122 | 소문자 끝 |
| <code>'0'</code> | 48 | 숫자 문자 시작 |
| <code>'9'</code> | 57 | 숫자 문자 끝 |

```c
char h = 'A';
printf("%c", h);  // 문자 출력: A
printf("%d", h);  // 숫자 출력: 65

/* 문자 연산 */

char a = 'F';
char b = 'A';

int val = a - b;    // 70 - 65
printf("%d", val);  // 5


/* 문자 비교 */
if ('F' > 'A') {
    printf("True"); // True
}

/* 대소문자 변환 */
char upper = 'F';
char lower = upper + 32;    // f

char lower2 = 'a';
char upper2 = lower2 - 32;  // A

/* 숫자 문자 변환 */
char c = '3';
int num = c - '0';  // 51 - 48 = 3
printf("%d", num);  // 3

/* ctype.h 함수*/
isupper(a)  // true
islower(b)  // false
isdigit(c)  // true
```
